# Atividade Aula 02

Este repositório contém atividades de JavaScript focadas em tipos primitivos, escopo léxico e tipos de referência.

## Arquivos

### atividade-aula02.js

**Como Rodar:**
```bash
node atividade-aula02.js
```

**O que Demonstra:**
Este arquivo demonstra os conceitos de tipos primitivos em JavaScript e escopo léxico em Node.js.

---

### tf03.js

**Como Rodar:**
```bash
node tf03.js
```

**O que Demonstra:**
Este arquivo demonstra práticas com tipos de referência (objetos e arrays) em JavaScript, incluindo:
- Criação e uso de objetos com propriedades simples
- Uso de arrays dentro de objetos
- Iteração em propriedades do objeto com `for...in`
- Iteração nos itens do array com `for...of`
- Comparação entre tipos de referência usando operador `===`
- Exemplo de cópia por referência
- Exemplo de cópia com spread operator
- Transformação do objeto em JSON com `JSON.stringify()`
